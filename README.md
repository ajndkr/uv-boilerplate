# uv-boilerplate

template repository for new uv projects with boilerplate code.

## usage

```shell
make run
```

to run project in a docker container:

```shell
make compose
```

## getting started

### pre-requisites

- [uv](https://docs.astral.sh/uv/#getting-started)

### init project

```shell
make init
```

to run pre-commit checks, run:

```shell
make ci
```
